# 🛒 Grocery MERN App

Project Live link->https://mern-stack-ecom-grocerywebsite.vercel.app

🔗 **GitHub Repository:**
https://github.com/sidharthhcj/mern-stack-ecom-grocerywebsite

A full-stack **Grocery E-commerce Web Application** built using the **MERN Stack**.
This project includes **user authentication, cart system, order management, seller dashboard, and forgot password functionality.**

---

## 🚀 Features

### 👤 User Features

* User Register & Login (JWT + Cookies)
* Forgot Password & Reset Password (Email based)
* Browse Products by Category
* Search Products
* Add / Remove items from Cart
* Update Cart Quantity
* Add & Select Delivery Address
* Place Orders (Cash on Delivery)
* View My Orders
* Secure Logout

---

### 🧑‍💼 Seller (Admin) Features

* Seller Login
* Add New Products
* View Product List
* Manage Orders
* Protected Seller Routes

---

## 🔐 Security

* Password Hashing using **bcrypt**
* **JWT Authentication**
* **HttpOnly Cookies**
* Secure Password Reset Token with Expiry
* Email based Password Recovery

---

## 🧰 Tech Stack

### Frontend

* React.js (Vite)
* React Router DOM
* Context API
* Tailwind CSS
* Axios
* React Hot Toast

### Backend

* Node.js
* Express.js
* MongoDB & Mongoose
* JWT
* bcryptjs

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/sidharthhcj/mern-stack-ecom-grocerywebsite.git
```

### 2️⃣ Install Dependencies

```bash
cd server
npm install

cd ../client
npm install
```

### 3️⃣ Run Project

```bash
npm run dev
```
